## install
```text
1. install this repository

curl -s https://packages.gitlab.com/install/repositories/gitlab/gitlab-ce/script.deb.sh | sudo bash

2. choose the exact version of gitlab to install 

sudo apt-get install gitlab-ce=12.8.7-ce.0

3. or wget the package and install manually

wget --content-disposition https://packages.gitlab.com/gitlab/gitlab-ce/packages/ubuntu/bionic/gitlab-ce_12.8.7-ce.0_amd64.deb/download.deb

Reference:
https://packages.gitlab.com/gitlab/gitlab-ce
https://packages.gitlab.com/gitlab/gitlab-ce/install
```
